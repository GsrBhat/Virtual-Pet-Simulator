# Virtual Pet Simulator 

A command-line based **Virtual Pet Simulator** written in Python.

You can feed, play with, and take care of a virtual pet by choosing options from a menu.  
The pet has **hunger** and **happiness** levels (0–100), which change based on your actions and time.

---

##  Objective

To practice core Python concepts like:

- Functions  
- Loops  
- Conditionals  
- Dictionaries  
- Basic game logic / state update

---

##  Features

- Feed your pet from available food items (inventory)
- Play with your pet using different toys
- Pet stats: `hunger` and `happiness` (0–100)
- Time passes and affects hunger and mood
- Game-over condition if pet is too hungry or too sad
- Simple, menu-driven console interface

---
